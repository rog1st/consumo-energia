## 📝 Sobre o Projeto
Este é um utilitário desenvolvido em **Python** para ajudar usuários a entenderem o impacto financeiro de seus aparelhos eletrodomésticos. O sistema solicita dados básicos de uso e fornece o custo estimado mensal na conta de luz.

### ✨ Diferenciais técnicos:
- **Tratamento de Input:** Aceita números com ponto ou vírgula (padrão brasileiro).
- **Validação de Dados:** Impede que o programa feche caso o usuário digite letras em campos numéricos.

---

## 🧮 Fórmula de Cálculo
O sistema utiliza a fórmula padrão de faturamento de energia:

$$Consumo (kWh) = \frac{Potência (W) \times Horas/Dia \times Dias/Mês}{1000}$$
$$Custo Total = Consumo (kWh) \times Valor do kWh$$
